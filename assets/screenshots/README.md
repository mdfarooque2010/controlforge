# Screenshots folder

Drop your real screenshots here, then update the corresponding `.screenshot-slot`
in `projectpulse.html` or `baseline-review.html` to use an `<img>` tag instead
of the placeholder text — the exact snippet to use is in an HTML comment
directly above each screenshot grid in those files.

Suggested naming (not required, just keeps things tidy):
- projectpulse/01-dashboard.jpg, 02-workbook.jpg, 03-delay-report.jpg
- baseline-review/01-health-check.jpg, 02-risk-flags.jpg, 03-report-output.jpg

Same files can be reused directly for your Microsoft Store listing screenshots
and for LinkedIn posts — no need to create separate versions.
