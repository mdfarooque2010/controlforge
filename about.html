Sheet AB-01 — Background

Built by the engineer who needed them first

I'm a Planning & Cost Control Engineer working in the water and EPC construction sector. I build these tools because I use Primavera P6 every day, and I got tired of doing the same manual reporting, baseline auditing, and schedule health checks by hand on every project.

Each tool here started as something I built to solve my own workload, then refined into something other planning engineers and PMO teams can use directly — without handing over their project data to a cloud service to get it.

Every tool in this set runs offline, on your own machine.
